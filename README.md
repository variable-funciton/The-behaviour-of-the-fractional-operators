# The-behaviour-of-the-fractional-operators

# Fractional Analysis Visualizer
This repository contains the Python script used for the numerical visualizations in the paper Composition of the Orlicz-Fractional Maximal Operators.

## Requirements
- numpy
- matplotlib
- sympy (for exact rational values and LaTeX export)
- A working LaTeX distribution (for `text.usetex=True`)
